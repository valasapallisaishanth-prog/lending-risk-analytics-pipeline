**Project Title
Lending Risk Analytics Pipeline
Project Overview**
This project builds an end-to-end lending risk analytics pipeline using machine learning, SQL, and data visualization to predict loan defaults and support credit decision-making.
Using historical lending data, the system performs data preprocessing, feature engineering, machine learning modeling, and borrower risk scoring to estimate the probability that a loan applicant will default.
The pipeline also simulates a production-style analytics workflow, where processed data is stored in a SQL database, risk predictions are generated, and insights are prepared for business dashboards.

**Key Features**
• Built a loan default prediction model using machine learning algorithms.
• Generated borrower default probabilities and credit risk scores.
• Implemented a risk segmentation framework (Low, Medium, High Risk).
• Created an automated loan approval recommendation system.
• Designed a SQL-based analytics layer for borrower segmentation and reporting.
• Prepared dashboard-ready datasets for business intelligence tools such as Tableau or Power BI
• Simulated a cloud-style data pipeline architecture inspired by AWS and Snowflake analytics workflows.

**Business Objective**
The goal of this project is to simulate how financial institutions evaluate loan applications by combining data analytics and machine learning to identify high-risk borrowers and improve lending decision workflows.
The system can help:
reduce default risk
prioritize manual loan reviews
automate approval recommendations
support data-driven lending strategies

**Dataset**
The project uses historical loan data from the LendingClub platform, which contains borrower financial information, loan characteristics, and repayment outcomes.
This dataset enables the development of predictive models for credit risk assessment and loan default prediction.\
